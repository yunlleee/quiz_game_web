# Quiz Game (2P)

This is a simple web-based quiz game designed for two players.  
Players take turns answering questions, and each correct answer increases their score.  
After all questions are completed, the final scores are displayed and a winner is announced.

## Features

- Two-player turn-based gameplay
- Score tracking for both players
- Multiple question types (multiple choice and short answer)
- Feedback shown after each answer
- Final results screen with winner display
- Option to restart the game or return to the menu

## File Structure

- `index.html` – main structure of the game interface
- `style.css` – styling and layout
- `js.js` – handles game logic such as questions, scoring, and turns

## How to Run

1. Make sure all files are in the same folder  
2. Open `index.html` in your browser  
3. Click **Start Quiz** to begin

## Technologies Used

- HTML
- CSS
- JavaScript

This project was created to practice basic web development concepts,
